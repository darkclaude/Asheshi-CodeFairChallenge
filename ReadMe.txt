Ashesi Code fair 2015
Team: Regent15
School: Regent University

Task: Find the minimal average of any slice containing at least two elements.
Programming Language:Php
Overview: 
(P, Q) = 0 ≤ P < Q < N

Algorithm:
slice (P, Q) = (A[P] + A[P + 1] +...+ A[Q]) / (Q − P + 1)

Requirements: Wamp server and Php
